使用说明: 

comment.js需要放在博客根目录下, 不要放在theme文件夹里

package.json最后几行的"script"脚本部分写入根目录package.json内, 以后提交的时候可以使用`npm run deploy`来自动化提交, 用`npm run debug`来自动化调试.
